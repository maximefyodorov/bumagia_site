Модульное оригами

На этой странице представлена наша коллекция наборов для создания фигурок в технике модульного оригами.
Каждый набор упакован в красочную коробку, в которой вы найдете:

  • Подобранные по цвету и количеству бумажные основы-прямоугольники для сворачивания базовых модулей
  • Детальную иллюстрированную инструкцию по сборке. Примеры инструкций можно найти здесь
Вся колекция разделена на две серии, их отличие — в общем количестве используемых для сборки фигурки модулей. Если их число меньше шестисот, то она имеет артикул с буквами ОМ, а сама коробка оформлена в красных тонах. Если же модулей используется больше шестисот, то буквы в артикуле будут уже другие — ОВ, а для коробки основным цветом будет синий.
